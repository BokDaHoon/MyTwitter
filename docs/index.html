SCRAP

타임라인의 글을 읽으면서 자신이 원하는 글이 있다면 해당 트윗을 스크랩할 수 있습니다. 스크랩한 트윗은 향후에 강력한 검색기능을 이용해서 이전 트윗을 쉽게 확인할 수 있습니다.

RECOMMEND SEARCH WORD

오직 트위터 사용자의 검색 트렌드로 검색어를 추천해주던 기존 방식에서, RT순위, 네이버, 다음 검색 순위 등으로 다양한 추천 검색어를 제공한다.

SCHEDULED TWEET

글을 올릴 때 즉시 업로드 되던 기존 방식에서 예약 전송 방식을 추가제공합니다. 이제 사용자는 원하는 시간을 지정하여 트윗을 올릴 수 있습니다.

My Twitter Feature

  • Tweet

    Twitter에 현재 생각, 추억 등을 올릴 수 있습니다. 즉시전송, 예약전송 모두 가능합니다.

  • Detail

    현재 Tweet에 세부 내용을 볼 수 있습니다.

  • Favorite

    사용자가 마음에 드는 트윗이 있다면, 마음에 든다는 표시를 할 수 있습니다.

  • Reply

    사용자가 마음에 드는 트윗에 댓글을 남기고 의견을 공유할 수 있습니다.

  • Timeline

    사용자가 팔로우한 사용자의 트윗들의 내용(글, 이미지, 트위터카드 등) 을 제공합니다.

  • Search

    사용자를 쉽게 검색할 수 있습니다. 또한 다양한 추천 검색어를 제공받을 수 있습니다.

  • Profile

    사용자가 올린 프로필 이미지, 트윗, 팔로워, 팔로잉 수 등 사용자의 내용을 볼 수 있습니다.

  • Scrap

    사용자가 원하는 글을 스크랩할 수 있고, 이전 스크랩 글을 쉽게 찾도록 검색기능을 제공합니다.
